## 技術スタック

### フロントエンド

- Fresh(Deno製フレームワーク)
- Tailwind CSS

### バックエンド

- Deno
- microCMS

### ホスティングサービス

- Deno Deploy

## サイト概要

### トップページ

トップページでは、サイト全体の概要と最新情報を確認できます。
サークルで進行中のプロジェクトや完了したイベント、今後のイベントなどまとめて表示しています。

### サークルについて

このページではサークルの基本情報や活動内容について説明しています。
本サークルの理念や活動の方向性やどのような活動をしているか、またサークルの歴史などを紹介しています。

### 記事

技術情報やサークルの活動内容、イベント報告など
活動に関する記事をまとめているページです。

### プロジェクト

サークル内で進行・完了したプロジェクトを紹介するページです。
現在開発中のプロジェクトや、過去に完了したプロジェクトがまとめられています。

### お問い合わせ

外部の方や新規メンバーなどがサークルに連絡できるページです。
入部希望やその他質問などを受け付けています。
